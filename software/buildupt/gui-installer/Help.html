v5.0 Installation Help v5.0 Installation Help

Welcome to Common Security Module's (CSM) User Provisioning Tool (UPT) v5.0 Installation Wizard online help. The Installation Provisioning Tool (UPT) v5.0 Installation Wizard online help. The Installation Wizard helps guide you through the installation and configuration process. This online help will assist you in entering the necessary configuration values and advancing through the wizard.

Each page of the Installation wizard contains four buttons, located at the bottom of each window:

Help: Opens this online help file

Previous: Returns to the previous page of the wizard

Next: Continues to the next page of the wizard

Quit: Exits the installation wizard

Details about each page of the wizard are provided below.

This software is developed by: NCI
Project Homepage is located at http://cabig.nci.nih.gov/tools/CSM

Release Notes

This page contains the Release Notes for the CSM release. Review the release notes to learn about the new features of the release, fixed bugs, known issues, and other useful information.

Licensing Agreements

This page contains the licensing agreement for using the CSM product. You are strongly encouraged to review the licensing information before proceeding.

By default the radio button to not accept the agreement is selected, and the Next button is grayed-out.

To install the product, you must select the I accept the terms of this licensing agreement radio button. This enables the Next button and allows you to continue with the installation.

Install or Upgrade Software

Use the Install or Upgrade radio buttons to select whether you are performing a new installation of the CSM UPT or upgrading an older version, respectively.

BE SURE to review the Warning posted on this page of the wizard. It informs you that if selecting Install, the installer will remove any previously installed application servers or databases, based on the information you provide in the wizard. If this warning applies to you, be certain to back up any application server and database instances if they exist. For more information, see the CSM page of the caBIG website: https://cabig.nci.nih.gov/tools/CSM.

Installation Type

Currently, the UPT Installer only provides for a Typical type installation. A Typical installation is a "Single Installation/Single Schema" deployment of CSM. If your deployment is Non-Typical, click Quit to exit the installer. You cannot use the Installation Wizard for any other type of deployment.

Installation Folder

Type in the path, or click Browse to browse for the folder where you want to install the CSM UPT. The JBoss Application Server will be installed in this folder.

If the directory you identify already exists, it will be overwritten. A message appears confirming that it is okay to overwrite the folder. You must click Yes on this message to continue. Clicking No returns you to the installer page where you can select a different installation location.

If you do not want to lose the contents of the existing folder, create a copy or backup of the folder before proceeding.

Select Server Components to Install

Identify which server components you want to install. Currently the only option available is to install JBoss and the Database Schema.

Upgrade Configuration

If you are using the installer to Upgrade your CSM UPT installation, the Upgrade Configuration page allows you to specify information about the existing UPT installation. Those items include:

Application Root Directory: Identify the existing location where UPT is installed. For example, C:\apps\upt

JBOSS Home: Identify the existing JBoss folder. For example, C:\apps\upt\jboss-4.0.5.GA

JBoss Server Name: Identify the name of the JBoss server. For example, default

Original CSM Schema Version: Identify the version of the existing CSM Schema (which will be migrated to the latest version). For example, 3.1

Choose Authentication Type

This page of the install wizard allows you to select the type of authentication you will be using. The Authentication type options are Database, LDAP, or LDAPS (also known as secure LDAP). Use the radio buttons to select one.

Depending on which Authentication Type you select, a different wizard page will appear, because the configuration details for each are different. Details on each of the authentication type configuration pages follows. Review the one that applies to your situation.

Installation Configuration Summary

The Installation Summary page provides a consolidated listing of the property information you provided to the installer via the wizard.

Review the Summary information closely! If you need to change anything about the information provided, use the Previous button to return to the wizard page where the information was entered.

If all of the information appears to be correct, click Next.

Installation Group Summary

The Group Installation Summary page simply identifies the group being installed at this time. Currently the only group that appears is csmupt.

Completing the Installation

The final pages of the install wizard show the progress of the installation and then upon completion, a page indicating that installation has completed successfully.

On the Installation Finished page, you can select to Generate an automatic installation script, which creates a script file containing your configuration information. This would allow you to reprocess the installation later, if you would need to do so for any reason (without having to go through the wizard again). The final wizard page also identifies the location of an uninstaller program, in the event you ever need to remove the CSM UPT from your machine.
